Install node js 
=============================

website: http://nodejs.org/

After installation:
Node was installed at: /usr/local/bin/node
Npm was installed at : /usr/local/bin/npm
Make sure that /usr/local/bin is in your $PATH.

check this via: echo $PATH

Install Bower
=============================

website: http://bower.io

Switch to administrator for installation and creation of new directories:
sudo su

Install Bower:
npm install -g bower

Stop sudo:
exit

Install Angular-filter (https://github.com/a8m/angular-filter):
bower install angular-filter
